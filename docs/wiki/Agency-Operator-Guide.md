# Agency Operator Guide

Wiki page for non-engineers to manage and review the meeting crew.

See [AGENCY_OPERATOR_GUIDE.md](../AGENCY_OPERATOR_GUIDE.md) for full screenshots and detailed instructions.

## Operational Checklist
1. Verify Meeting UUID is entered correctly.
2. Select enabled specialists.
3. Configure the confidence slider.
4. Execute analysis and view live trace steps.
5. Perform manual approval or retry escalated steps if blockers arise.
