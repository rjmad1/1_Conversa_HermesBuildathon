## Excluded Modality Scope (Release Out-of-Scope)

The following requirements are explicitly excluded from this release. Any implementation of these components represents a violation of the milestone isolation scope.

| ID | Modality | Excluded Description | Verification Method | Expected Boundary behavior | Notes |
|---|---|---|---|---|---|
| **AGA-EXC-VID** | Video | Ingestion, processing, recording, or analysis of video | Code & API check | `POST /api/v1/meetings/:id/video` returns `404`/`503` or rejected with `415` | Handled via `MEDIA_VIDEO_ENABLED=false` |
| **AGA-EXC-CAM** | Camera | Camera and webcam permission prompts in the UI | Browser console check | No `navigator.mediaDevices.getUserMedia` calls in UI source | Camera hardware never requested |
| **AGA-EXC-MIC** | Microphone | Browser microphone live streaming | Browser console check | Live audio streaming is out of scope; files only | wispr/ElevenLabs streaming features deferred |
| **AGA-EXC-INT** | Integrations | Direct pipeline writes to Jira, Salesforce, GitHub, Slack | Integration check | Proposed actions list shown in UI; export button outputs JSON only | No third-party network outbound writes |
| **AGA-EXC-KEY** | Auth Keys | Browser-entered API keys stored persistently on server | DB inspect | Keys processed strictly ephemerally in memory | Ephemeral BYOK never saved in database |
