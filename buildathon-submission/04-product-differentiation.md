# Product Differentiation

Conversa differentiates on **controlled actionability** rather than summary quality alone.

## Key Differentiators (current)
1. **Meeting-to-action orientation**
   - Output is explicitly action-structured, not just narrative summaries.
2. **Human-in-the-loop governance**
   - Approve/reject is mandatory in the core flow.
3. **Tenant/workspace isolation behavior**
   - Cross-scope reads/mutations are denied in adversarial/smoke evidence.
4. **Non-disclosing secure response behavior**
   - Wrong-scope access returns not-found style behavior.
5. **Structured audit events**
   - Workflow state transitions are auditable.
6. **Provider abstraction**
   - Transcription/analysis interfaces support fake + OpenAI paths.
7. **Runtime-safe structured logging + redaction**
   - Security narrative includes recursive redaction and portability hardening.

## Not claimed as current differentiators
- Autonomous external enterprise execution.
- Production SSO / enterprise identity.
- Durable multi-region persistence.
- Fully live verified Jira/Slack/Salesforce transactions.
