# Three-Minute Demo Script

## 0:00–0:20 — Problem
"Teams make decisions in meetings, but execution gets lost in notes, chat, and manual re-entry. Conversa closes that gap by converting conversations into governed, reviewable actions."

## 0:20–0:40 — Positioning
"Conversa is a meeting-to-action prototype: transcript in, structured decisions/risks/actions out, with mandatory human approval and audit evidence."

## 0:40–1:30 — Transcript-to-analysis
"I’ll use a synthetic transcript. We submit it, run analysis, and Conversa returns a summary, key decisions, risks, and proposed actions with owners and due dates."

## 1:30–2:10 — Approval + audit
"Now we approve one action and reject one action with reason. Then we open audit history and confirm event traceability."

## 2:10–2:35 — Security evidence
"Cross-tenant and cross-workspace reads/mutations are denied. We validated this with tenant isolation tests, an adversarial runner, and smoke checks."

## 2:35–3:00 — Vision + close
"Today is a governance-first prototype. Next milestone is pilot hardening: production identity, durable storage, and one verified live integration."
"Conversa helps teams turn meeting conversations into accountable execution."
